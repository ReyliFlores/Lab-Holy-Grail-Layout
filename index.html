The Header provides the tools for navigation and sets the theme for the website. The Header element is by default at a static positioning. This means that the element is positioned to the normal flow of the document and it wont move relative to other elements. This element can only be moved if padding or margin CSS code is applied.
The nav element is at a fixed position which means when scrolling, the elemnt will remain in its place. The CSS code such as top:0 and left:0 ensures that the lement stays at the top left corener of the web page.
The articles element positioning is relative so that the element can be displaced at left:30px and top:45px. The relative positioning allows for desplcaement of any element.
This is just another article element that is just related to the fisrt article element. The only difference is that it is shifted from the top by 45 pixels.
For the aside element, it uses the absolute positioning element so that it removes itslef from the normal flow of the documnet. It can be displaced while using the nearest element and in this case, it would be the articles element.
The footer also has static positioning just like the header. Instead of the footer being at the top left of the page like the header, it will remain at the bottom of the page.
